## 📊 Activity Scoring Algorithm

### There-Then Multi-Factor Scoring (40/20/20/20 weights)
```javascript
FINAL_SCORE = (
  0.40 * weatherScore +      // Weather conditions
  0.20 * airQualityScore +   // Air quality
  0.20 * timeScore +         // Optimal timing (solar data)
  0.20 * affinityScore       // User preferences
) * ageAdjustment            // Age-based filtering
```

### Age-Based Adjustments
- **Adult-only activities** (bars, nightclubs, casinos): `× 0` when kids < 21
- **Kid-friendly activities** (playgrounds, zoos): `× 1.2-1.3` when kids < 13
- **Group affinities**: Simple average across all trip participants

## 🛠️ Critical Bug Fixes This Session

### 1. **Firestore Undefined Values Error**
- **Problem:** `updateDoc()` called with undefined values (userId, age)
- **Fix:** Only add optional fields if they exist (lines 315-321 in TripDetail.tsx)
- **Result:** Members without registered accounts can now be added to trips

### 2. **Firestore Permissions Error**
- **Problem:** Security rules checked `resource.data.ownerId` which fails on non-existent docs
- **Fix:** Changed to `cirqleId == request.auth.uid` (works for both cases)
- **File:** `firestore.rules` line 191

### 3. **UTC Timezone Display**
- **Problem:** Sunrise/sunset showing UTC times instead of local timezone
- **Fix:** Return ISO timestamps from Cloud Function, convert in frontend
- **Result:** Times display in user's local timezone (EST, PST, etc.)

### 4. **inviteToCirqle Function 500 Error**
- **Problem:** `cirqleDoc.exists()` called as function instead of property
- **Fix:** Changed to `cirqleDoc.exists` (line 1650 in functions/index.js)
- **Result:** Email invites now work correctly
